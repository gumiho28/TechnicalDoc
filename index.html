HTML Introduction

HTML stands for HyperText Markup Language. It is used to structure a web page and its content.

HTML elements include various tags like <h1>, <p>, <a>, and more.

These tags structure content on the page.

HTML is the foundation of any web page, working alongside CSS and JavaScript.

<h1>Hello World</h1>
